jQuery Dialogr 2.1

Maximize. Minimize. Restore.

This plugin is a modified copy of jQuery UI Dialog widget, adding minimize, restore and maximize buttons.

Open Dialogr 1 Open Dialogr 2 Open Dialogr 3

Usage

Well, use it as any normal dialog, but with one extra "r" at the end:

$(#dialog).dialogr();

You can use it with all jQuery UI Dialog options!

But jQuery Master Dialog adds two other options (when false the button doesn't appear):

$("#dialog").dialogr({maximized:false,minimized:false});

Off course, without minimize and restore buttons, it'll be the original UI Dialog again.

Known Bugs

1) Variables window.minimized and window.maximized should be stored at Dialogr instance, not at window object.

2) If you minimize the Dialogr and then close it, when you open it again the content height isn't calculated correctly.

Fixed Bugs

1) Passed options for Dialogr were not working at all. Fixed by Joseph Knapp
